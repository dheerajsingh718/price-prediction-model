Project Dependencies
====================

Core Python Packages
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- statsmodels
- scipy
- openpyxl

Additional Packages Used In Notebooks
- holidays
- lightgbm
- colorama

Install (pip)
- pip install pandas numpy matplotlib seaborn scikit-learn statsmodels scipy openpyxl holidays lightgbm colorama

Notes
- openpyxl is required for reading/writing .xlsx files with pandas.
- lightgbm is used in model training cells.
- colorama appears in EDA notebook imports.
